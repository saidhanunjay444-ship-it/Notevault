About NoteVault

📚

NoteVault

Version 1.0.0 •March 2026

Designed &Developed by Project K Squad

NoteVault is a college notes sharing platform designed and developed by Project K Squad . Each note focuses on one subject with a detailed list of topics covered. Built with Trie-based search, HashMap indexing, MergeSort ordering, IndexedDB file storage, and full OOP architecture.

Key Features

  • Each note covers one subject with a detailed list of topics inside it
  • Upload and contribute your own notes — files stay in your browser only
  • Download pre-loaded notes directly from the files folder on the server
  • Smart filters that update in real-time as you select different options
  • Start typing and get instant search suggestions powered by Trie data structure
  • Bookmark your favorite notes for quick access anytime
  • Sort notes by semester, course, year, or alphabetically with one click
  • Dark mode support, 15+ keyboard shortcuts, and fully responsive design

Architecture

DSA Trie, HashMap, HashSet, Queue, MergeSort, BinarySearch, Levenshtein
OOP Observer, Factory, Singleton, Strategy, Template, MVC
Stack Semantic HTML5, CSS3 Custom Properties, ES6+ Vanilla JS
Storage IndexedDB for files, localStorage for settings, import/export JSON
Files Server files from files/ directory, contributed files in IndexedDB

Privacy &Security

Your Privacy Matters

NoteVault is a fully client-side application built by Project K Squad. No data ever leaves your browser.

No Data Collection

NoteVault does not collect, transmit, or store any personal information on external servers. Everything runs entirely within your web browser.

Local Storage Only

Your bookmarks, preferences, and settings are stored in localStorage. Contributed files are stored in IndexedDB. No cookies are used.

  • localStorage: Bookmarks, theme, view settings, recent searches, contributed note metadata
  • IndexedDB: Uploaded file data from contributions — stored locally in your browser
  • No Cookies: NoteVault uses zero cookies
  • No Analytics: No tracking scripts, pixels, or analytics tools

File Downloads

Pre-loaded notes are served from the local files/ directory. Contributed files are retrieved from your browser's IndexedDB storage.

Security Measures

  • Input Sanitization: All user inputs are sanitized to prevent XSS attacks
  • Content Security: No external scripts or resources are loaded
  • Browser Sandbox: All operations run within the browser's security sandbox
  • No Network Requests: Zero external API calls or network requests
  • File Validation: Uploaded files are validated for type and size

Your Rights

  • Export: Export all your data as JSON anytime from Settings
  • Import: Import previously exported data to restore your setup
  • Delete: Clear bookmarks, contributed files, or all data from Settings
  • Portability: Your exported JSON file works across any browser

Last updated: March 2026. This privacy policy applies to NoteVault v1.0 by Project K Squad.
